# FitPro Station — Simulador Financiero

Herramienta profesional de costeo, fijación de precios y simulación de escenarios financieros para FitPro Station.

## Características

- 📊 **Dashboard Estratégico** — Métricas en tiempo real con perfil de ventas configurable
- 🧂 **Gestión de Ingredientes** — CRUD completo con cálculo automático de costo por unidad
- 📦 **Productos** — Recetas compuestas con ingredientes y calculadora de precios bidireccional
- 🏢 **Costos Fijos** — Registro y categorización de gastos operativos
- 🔬 **Simulador 3 Modos** — Goal Seek, Crisis/Ajuste, y Mix de Ventas

## Stack Tecnológico

- HTML + CSS + JavaScript (Vanilla)
- [Supabase](https://supabase.com) como base de datos en la nube
- Diseño dark theme premium con tipografía Inter

## Cómo Usar

1. Clona este repositorio
2. Abre `index.html` en tu navegador o usa un servidor local:
   ```bash
   npx http-server . -p 8080
   ```
3. Los datos se sincronizan automáticamente con Supabase
